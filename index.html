Marvel Cinematic University (MCU) API

This is a Marvel Cinematic Universe (MCU) API server that provides information on character's in Marvel's movies and tv shows. For this API, there is no auth key needed. I utilized a CSV to JSON converter to convert data from a Kaggle dataset which sourced from a dataset at Kamis Data Program Indonesia. All characters are owned and copywrite by Marvel. This API documentation was written via Swagger at http://swagger.io.
